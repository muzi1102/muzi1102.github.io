css的伪元素

发表于 2015-12-05   |   分类于 css/css3

前言

之前，我总是分不清伪元素和伪类，还好现在能分清了，不然，也不敢写这篇文章。在这里我们主要是记录下伪元素:after和：before。在Web页面的HTML源代码里，你实际上找不到它们，但从视觉上，你却能看到它们的存在，所以我们用这个“伪”字就是表示它们假元素。

伪类和伪元素

  • 伪类用于向某些选择器添加特殊的效果。

  • 伪元素用于将特殊的效果添加到某些选择器。

    :before和:after

    :before会在真正元素之前插入一个额外的元素,：after会在真正元素之后插入一个额外的元素，代码胜过一切言语，talk is cheap,show me the code

    我们是共产主义接班人

伪元素也是元素，我们可以对其设置css样式，这里就不做演示，毕竟简单，注意一点，如果要对这些元素设置宽高就要设置display:block/inline-block，伪元素也可以和伪类混合使用
