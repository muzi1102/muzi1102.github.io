正文

数组的创建

创建数组的方法有2种

数组直接量

使用数组直接量是创建数组最简单的方式，在方括号中用逗号隔开即可。例如：

var a=[];
var b=[1,2]

数组直接量中的值不一定要是常量，它可以是任意的表达式，还有如果省略数组直接量中的某个值，省略的元素会被赋予undefined;

调用构造函数Array()

这是创建函数的另一种方式，有3中调用方式

var arr1=new Array()//无参数
var arr2=new Array(10)//指定长度
var arr3=new Array(1,2,3)

这种方式显然没有第一种简单(^o^)/

数组的读写

使用[]可以用来访问数组中的元素，同时该语法也可以向数组写入元素。
还有一个方法push()可以向数组添加元素

数组的长度

每一个数组都有一个length属性。它代表数组的元素的个数，它的值比数组最大的索引大1
当我们把数组的长度设为小于当前长度会怎么样呢

var a=[1,2,3,4,5]
a.length=3 //a[1,2,3]
a.length=0 //删除所有元素

在es5中，我们可以用Object.defineProperty()让length属性变为只读的。那么上面那种方式就无效。

数组的添加和删除

数组的添加

var a=[]
    a[1]='aa'//这是最简单的方式
a.push('bb');//在数组的尾部添加元素可以是一个或者多个
a.[a.length]//这种是向尾部添加一个元素
a.unshift('111')//这是向元素的首部添加一个元素

数组的删除

var a=[1,2,3]
var b=[23,5,6]
delete a[1]//删除第2个元素
a.length //3。。这种删除方式并不会影响数组的长度
b.shift();
b.pop();

使用delete不会改变数组的长度，并且也不会把元素从高索引移下填充已经删除的元素。那么用delete删除元素的数组就变成一个稀疏数组了。。但是使用shift()删除元素就会元素从高索引移下填充已经删除的元素
