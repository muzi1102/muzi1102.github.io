flip

发表于 2015-11-27   |   分类于 css/css3

前言

之前对css3了解一直停留在我大概知道css3有哪些属性，用得比较多的是border-radius,box-shadow等等，好像对animation,transfrom,transition。没有深入了解，导致现在我也说不清，他们如何用才好，总是弄混，觉得它们长得好像。哈哈哈，我对自己无语/(ㄒoㄒ)/~~。这几天想用css3写一个翻转效果(flip),只好去查资料。再自己写

—————-分不清的分割线————-

####一：写结构 ####
第一步就是写html啦，先把这个翻转效果的整体结构写出来

<div class="card effect__hover">
    <div class="card__front">
        <span class="card__text">front</span>
    </div>
    <div class="card__back">
        <span class="card__text">back</span>
    </div>
</div>

####二：写css ####

首先给包裹div设置宽高，并且position:relative,让它的子元素以它为参照进行定位

.card {
    position: relative;
    padding-bottom: 25%;
    width: 25%;
    text-align: center;
}

二个孩子节点的样式

.card__front,
.card__back {
position: absolute;
top: 0;
left: 0;
width: 100%;
height: 100%;
-webkit-backface-visibility: hidden;
backface-visibility: hidden;
-webkit-transition: -webkit-transform 0.3s;
transition: transform 0.3s;
}

backface-visibility: hidden;这是隐藏旋转div的背面

.card__front {
    background-color: #ff5078;
}
.card__back {
    background-color: #1e1e1e;
    -webkit-transform: rotateY(-180deg);
    transform: rotateY(-180deg);
}

背面的div沿Y轴旋转-180deg,正好让我们看不见

.card__text {
    display: inline-block;
    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    margin: auto;
    height: 20px;
    color: #fff;
    font-family: "Roboto Slab", serif;
    line-height: 20px;
}

写到这里基本上就快完成啦，只要在鼠标移到div上，把现在看到div和看不到的div都沿Y轴旋转180deg(正负无所谓)

.card.effect__hover:hover .card__front {
    -webkit-transform: rotateY(180deg);
    transform: rotateY(180deg);
}
.card.effect__hover:hover .card__back {
    -webkit-transform: rotateY(0);
    transform: rotateY(0);
}

###结束语####
以上就完成了一个简单的css3旋转的效果
