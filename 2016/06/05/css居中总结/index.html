css居中总结

发表于 2016-06-05   |   分类于 css/css3

前言

之前，做baiduife的html的题目时。好像有一道css居中的联系。这个居中方法一直知道，但是欠缺总结。

水平居中

行内元素

 nav {
    text-align: center;
    background: white;
    margin: 20px 0;
    padding: 10px;
}

nav a {
    text-decoration: none;
    background: #333;
    border-radius: 5px;
    color: white;
    padding: 3px 8px;
}
<header>
    This text is centered.
</header>
<nav role='navigation'>
    <a href="#0">One</a>
    <a href="#0">Two</a>
    <a href="#0">Three</a>
    <a href="#0">Four</a>
</nav>

ps:主要是用到了text-align: center;

块级元素
main {
background: white;
margin: 20px 0;
padding: 10px;
}

.center {
    margin: 0 auto;
    width: 200px;
    background: black;
    padding: 20px;
    color: white;
}
 <main>
    <div class="center">
        I'm a block level element and am centered.
    </div>
</main>

ps:主要使用margin:auto（这个居中的前提是已知需要居中元素的宽度）

不止一个块级元素的水平居中
body {
background: #f06d06;
font-size: 80%;
}

main {
background: white;
margin: 20px 0;
padding: 10px;
}

main div {
background: black;
color: white;
padding: 15px;
max-width: 125px;
margin: 5px;
}

.inline-block-center {
text-align: center;
}

.inline-block-center div {
display: inline-block;
text-align: left;
}

.flex-center {
display: flex;
justify-content: center;
}
<main class="inline-block-center">
<div>
I'm an element that is block-like with my siblings and we're centered in a row.
</div>
<div>
I'm an element that is block-like with my siblings and we're centered in a row. I have more content in me than my siblings do.
</div>
<div>
I'm an element that is block-like with my siblings and we're centered in a row.
</div>
</main>
<main class="flex-center">
<div>
I'm an element that is block-like with my siblings and we're centered in a row.
</div>
<div>
I'm an element that is block-like with my siblings and we're centered in a row. I have more content in me than my siblings do.
</div>
<div>
I'm an element that is block-like with my siblings and we're centered in a row.
</div>
</main>

ps:这里第一个使用了（display：inline-block+text-align）第二个使用了flexbox的布局

垂直居中

行内元素
元素只有一行的情况
main {
background: white;
margin: 20px 0;
padding: 50px;
}

main a {
    background: black;
    color: white;
    padding: 40px 30px;
    text-decoration: none;
}
<main>
    <a href="#0">We're</a>
    <a href="#0">Centered</a>
    <a href="#0">Bits of</a>
    <a href="#0">Text</a>
</main>

ps:这里可以使用2种方法。第一种让他们的padding-top和padding-bottom相等。第二种是设置和高度相等的行高

多行的行内元素
.center-table {
display: table;
height: 250px;
background: white;
width: 240px;
margin: 20px;
}

.center-table p {
    display: table-cell;
    margin: 0;
    background: black;
    color: white;
    padding: 20px;
    border: 10px solid white;
    vertical-align: middle;
}
 <div class="center-table">
    <p>I'm vertically centered multiple lines of text in a CSS-created table layout.</p>
</div>

ps:这里可以使用table布局，也可以使用css的display的table和table-cell。但是必须在子元素中加上vertical-align: middle;
使用table布局的则不用，因为是默认的。毫无疑问，这里可以使用flexbox布局

块级元素

已知元素的高度
PS:首先。设置元素的top为50%。在设置margin-top为自身高度的一半

不知道高度

.parent {
      position: relative;
}
.child {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}

PS:可以用css3的transform属性达到目的。。当然。可以使用flexbox布局

水平垂直居中

其实这里和垂直居中大同小异。只是加入了水平方向的处理

第一种情况是一直宽高

.parent {
  position: relative;
}

.child {
  width: 300px;
  height: 100px;
  padding: 20px;

  position: absolute;
  top: 50%;
  left: 50%;

  margin: -70px 0 0 -170px;
}

第二种是不知道宽高

.parent {
  position: relative;
}
.child {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

css3的tranform属性
最后可以flexbox
