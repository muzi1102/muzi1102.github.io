css layouts

发表于 2016-06-08   |   分类于 css/css3

前言

今天看到一个说css的品字布局，所以，想记录下css的一些布局，怕自己忘记

品字

一个满屏 品 字布局 如何设计，使用float和inline

.top{
    width:100%;
    background-color:red;
    height:50px;
}
.left{
    width:50%;
    background-color:#000;
    /* display:inline-block; */
    float:left;
    height:50px;
}
.right{
    width:50%;
    background-color:yellow;
    /* display:inline-block; */
    float:left;
    height:50px;
}
<div class="container">
    <div class="top"></div>
    <div class="left"></div>
    <div class="right"></div>
</div>

PS:这里要注意的是使用inline-block。可能要知道怎么去掉元素的间隙。使用float要记得清楚浮动
同理。倒品字的布局可得

多列布局

左列定宽，右列自适应

法一：

<div class="box">
    <div class="left"></div>
    <div class="right"></div>
</div>
.left{
    width:200px;
    height:100px;
    background-color:red;
    float:left;
}
.right{
    /* overflow:hidden; */
    margin-left:200px;
    height:100px;
    background-color: blue;
}

ps：使用float+margin。还可以用overflow:hidden替换margin达到同样效果。overflow的原理是BFC。

法二：
.left {
background-color: green;
float: left;
width: 220px;
margin-right: -100%;
height: 100px;
}

.content {
    float: left;
    width: 100%;
}

.contentInner {
    margin-left: 220px;
    height: 100px;
    background-color: orange;
}
<div class="left">
</div>
<div class="content">
    <div class="contentInner">
    </div>
</div>

这个和上面那个还蛮像哈。还有很多其他的方法如flexbox

两列等高布局

这个布局平时还真没怎么注意。sad

 .container{
        overflow: hidden;
    }
    .left{
        float: left;
        width: 150px;
        background-color: red;
        padding-bottom: 2000px;
        margin-bottom: -2000px;
    }
    .right{
        float: left;
        width: 450px;
        background-color: blue;
        padding-bottom: 2000px;
        margin-bottom: -2000px;
    }
<div class="container">
    <div class="left">我是left</div>
    <div class="right">我是right我是right我是right我是right</br>我是right我是right我是right我是right</div>
</div>

需要了解更多的等高布局可以查看http://www.w3cplus.com/css/creaet-equal-height-columns

两列定宽，一列自适应

这个简单可以使用(float+margin/overflow)等。这里说明使用flexbox

.left1,.left2{
    width:200px;
    height:100px;
    background-color:red;
    float:left;
}
.left2{
    background-color:blue;
}
.left3{
    background-color:red;
    flex:1;
    height:100px;
}
.box{
    display:flex;
}
<div class="box">
<div class="left1"></div>
<div class="left2"></div>
<div class="left3"></div>
</div>

两侧定宽，中栏自适应

这个解决办法和上面的方法相似。这里就不贴代码了。

一列不定宽，一列自适应

        display:table;
    }

   }

    <div class="right">哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈哈</div>
</div>

这里使用float+overflow/flexbox/table-cell都可以实现

多列等分布局

.column {
    float: left;
    width: 25%;
    padding-left: 20px;
    box-sizing: border-box;
    border: 1px solid red;
}
<div class="parent">
    <div class="column">1</div>
    <div class="column">1</div>
    <div class="column">1</div>
    <div class="column">1</div>
</div>

使用float实现/利用table实现/利用flex实现都可以实现
