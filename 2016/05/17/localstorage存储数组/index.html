localstorage存储数组

发表于 2016-05-17   |   分类于 javacript

前言

说之前，先来说一说html5的本地存储，提供了一种方式让网站能够把信息存储到你本地的计算机上，并再以后需要的时候进行获取。这个概念和cookie相似，区别是它是为了更大容量存储设计的。

cookie:主要用途有保存登录信息，比如你登录某个网站市场可以看到“记住密码”。它的大小限制为4kb左右。并且每次你请求一个新的页面的时候cookie都会被发送过去。

localStorage和sessionStorage：HTML5的storage是存储在你的计算机上，网站在页面加载完毕后可以通过Javascript来获取这些数据。

localStorage和sessionStorage的区别
它们俩非常相识。唯一的不同是：localStorage存储的数据没有生命周期，除非用户自己删除，否则会一直存在。但sessionStorage，当页面关闭后，sessionStorage的内容会被删除。

查看：Chrome和Firefox的调试工具查看Storage。Chrome下打开控制台
点击Resources。左侧可以看见localStorage和sessionStorag的选择按钮

localStorage的使用

这里说说localStorage的使用：

  • 首先自然是检测浏览器是否支持本地存储

    if (window.localStorage) {
       alert('This browser supports localStorage');
} else {
    alert('This browser does NOT support localStorage');
}

    你也在caniuse上检查浏览器的支持情况。

  • 存储数据

       localStorage.setItem("lastname", "Smith");
localStorage.lastname = "Smith";

  • 删除数据

    localStorage.removeItem("lastname");

  • 取数据

    localStorage.setItem("lastname");
localStorage.lastname;

localStorage存数组

localStorage是以字符串的形式存储数据的。

var arr=[1,2,3,4]
localStorage.arr=arr;
console.log(localStorage.arr);

去控制台查看就车祸了。因为打印出来的数据根本就不是一个数组。那要怎么做呢？这就不得不提JSON.stringify()和JSON.parse()啦

var arr=[1,2,3,4]
var str=JSON.stringify(arr);
localStorage.arr=str;
// 取数据
var arr1=localStorage.arr;
    result=JSON.parse(arr1)
console.log(result);

同样JSON数据和对象都可以这么使用

这里顺便写下js的二维数组的声明
首先js没有直接声明二维数组的方法。二维数组就是一维数组的数组

var board=new Array();//一维数组
for(var i=0;i<4;i++){
    board[i]=new Array();//一维数组的数组
    for(var j=0;j<4;j++){
        board[i][j]=0;
    }
}
