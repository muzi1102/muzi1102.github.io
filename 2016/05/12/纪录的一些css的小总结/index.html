display和visibility

display:none和isibility：hidden都可以让元素隐藏，那么它们有什么不同呢？先看代码，用代码说话

<style>
     .none{
        border: 1px solid red;
        margin-bottom:5px;
    }
    .visibility{
        border: 1px solid red;
    }
    p{
        background-color: #F9FA0A;
    }
    .dis{
        display:none;
    }
    .visi{
        visibility: hidden;
    }
</style>
<div class="none">
    <p>display:block</p>
    <p class="dis">display:none</p>
</div>
<div class="visibility">
    <p>visibility:visible</p>
    <p class="visi">visibility:hidden</p>
</div>

运行完这个小例子之后，发现问题就很简单了display:none之后，元素不见了，浏览器不会在为它保留空间。但是visibility:hidden，浏览器会为她保留空间。。不会让它的位置被其他元素占用，这个标签会被渲染
这里其实会涉及到一个对性能很重要的东西，重排和重绘http://kb.cnblogs.com/page/169820/

input的不支持伪元素

这点事有一天写一个小东西发现的，，当时被坑到了，一脸懵逼。
网上的解释是【一个元素文档树内容之前和之后的内容】就是指这个元素是要可以插入内容的，也就是说这个元素要是一个容器。input,img,iframe等元素不能包含其他元素，所以不能通过伪元素插入内容。但是在chrome中radio和checkbox貌似有效，，那么问题来了，要怎么才能让它们有伪元素的效果了。

<style>
    div:after
    {
        content: "hah";
    }
    span:after{
        content: "hah";
    }
</style>
<div><input type="text"/></div>
<input type="text"/><span></span>

这里就ok，简而言之，借腹生子。。2333

box-sizing

这是一个非常好玩的css属性,同样看代码

<style>
    .init{
        width: 100px;
        border:1px solid #000;
        height: 20px;
        padding: 10px;
        margin-bottom:3px;
        margin:5px;
    }
    .borderBox{
       box-sizing: border-box;
    }
    .contentBox{
        box-sizing: content-box;
    }
<stlye>
<div class="boxSizing">
    <div class="init"></div>
    <div class="init borderBox"></div>
    <div class="init contentBox"></div>
</div>

content-box:宽度和高度分别应用到元素的内容框。在宽度和高度之外绘制元素的内边距和边框。

border-box:为元素指定的任何内边距和边框都将在已设定的宽度和高度内进行绘制。通过从已设定的宽度和高度分别减去边框和内边距才能得到内容的宽度和高度。

元素布满整个屏幕

做一个遮罩层会让元素铺满整个屏幕了，那么就不可缺少这个东西了

<style>
    left: 0;
    top: 0;
    bottom: 0;
    right: 0;
    background-color: #ccc;
    position: absolute;
    overflow: hidden;
</style>
<div class="fullpage"></div>

You can make 100% width and 100% height using top: 0; left: 0; bottom: 0; right: 0;
利用它还可做一个小东西

`.wrapper{
        display: block;
        position: fixed;
        z-index: 999;
        right: 10px;
        top: 10px;
        width: 40px;
        height: 40px;
        background-color: rgba(0, 0, 0, 0.25);
    }
    .arrow{
        font-size: 20px;
        height: 0.1em;
        width: 1.4em;
        background: white;
        position: absolute;
        left: 0;
        right: 0;
        top: 0;
        bottom: 0;
        margin: auto;
    }`
`<div class="wrapper">
    <div class="arrow"></div>
</div>`
