input的type='file'

发表于 2016-04-09   |   分类于 Html5

前言

今天用到input的type=’file’,发现自己对这个属性了解太少。于是决定好好查找资料。由于我记性差劲，怕忘记，就有的这篇文章。

选择文件

html5新增了File API,可以让网页要求用户选择本地文件，并且读取这些文件的信息了。选择的方式既可以是HTML的input type=’file’元素，也可以是拖拽 。

<input type="file" id="input">

在选择文件的时候，可以一次选择一个或则多个，如果允许用户上传多个文件，记得要在input中加入multiple 。可以打印节点的files的属性知道。所选择的这些对象被包围在一个FileList对象中。它是一个类数组的形式。

属性

accept:Sets or returns the value of the accept attribute of the file upload button

autofocus:Sets or returns whether a file upload button should automatically get focus upon page load

files：Returns a FileList object that represents the file or files selected with the file upload button

multiple：Sets or returns whether a user is allowed to select more than one file in the file upload field

获取所选文件的信息

用户所选择的文件都存储在了一个FileList对象上，其中每个文件都对应了一个File对象。你可以通过这个FileList对象的length属性知道用户一共选择了多少个文件,每一个File对象都有三个属性：

name：
文件名,只读字符串,不包含任何路径信息.

size：
文件大小,单位为字节,只读的64位整数.

type：
MIME类型,只读字符串,如果类型未知,则返回空字符串.

var inputElement = document.getElementById("inputField");
console.log(inputElement.files.length);
for(var i =0 ;i<inputElement.files.length;i++){
    console.log(files[i].name+''+files[i].size+''+files[i].type)
}
