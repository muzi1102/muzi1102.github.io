atom 的介绍

在前端的代码编辑器中就我所了解的就有好几种，如sublime,webstrom,EditPlus、NotePad++等等。我们今天要介绍的是由Github的程序员们创造的。他们使用目前最先进流行的技术重新打造了一款称为“属于21世纪”的代码编辑器。atom和sublime非常的相似。但是atom是开源的，社区活跃，支持支持插件扩展，可配置性非常高。对于追求主流技术的码农是个非常不错的选择。

安装

Atom 代码编辑器支持 Windows、Mac、Linux 三大桌面平台，完全免费。直接去官网下载即可。

基本功能介绍及使用

1：主题选择

在atom的welcom界面选择choose a Theme,这里的主题分为ui(整个界面的颜色)和Syntax(代码的颜色)。
我选择的是atom dark。因为我觉得和我之前使用sublime主题相似，有亲切感。编辑器自带几种选择，你要是对这几种都不满意。可以去下载相应主题插件

2：快捷键

这里介绍windows。其他系统自行查找

1：ctrl + shift + p 启动，通过命令输入框可以使用任何插件（自带插件和第三方插件）的功能

2：ctrl+n 新建一个文件（这是我用的最多一个快捷键）

3：ctrl+f 查找关键字，退出按esc即可

4：ctrl+g 光标跳转到相应的行

5：ctrl+l 快速选中一行

6：ctrl+d 快速选中相同单词

3：多屏写代码

在navbar的View->panel。选择你要的分屏方式。。想怎么分就怎么分啊

atom安装插件

1:快捷键ctrl+，或者flie->settings。

2:点击Install,

3：在 Install Packages 下面。选中你要安装的插件（也可以是搜索的结果）。是不是非常简单。我觉得比sublime都好用

好用的插件

atom只所以这么好用是因为它有强大的插件支持。开源，社区活跃。插件层出不穷。这几我来介绍几个我用的插件。正因为这几个插件，我对atom充满好感

强烈推荐

1：emment:这是一个神器，快速编码。用sublime text的同学都知道这个插件，我这里就不多说了，如有不了解的，请戳http://www.w3cplus.com/tools/emmet-cheat-sheet.html

2： javascript-snippets。js的代码补全插件，非常好用。详细介绍请务必戳https://atom.io/packages/javascript-snippets这里详细介绍了简写的命令。愉快编码的必备啊

3：highlight-selected 这是文档的介绍Double click on a word to highlight it throughout the open file.双击高亮选中所有单词。详细请戳https://atom.io/packages/highlight-selected

4：atom-beautify，(快捷键ctrl+alt+b)这个对于写代码不注重格式的人非常好。美化你代码的格式。这是官方文档介绍：Beautify HTML, CSS, JavaScript, PHP, Python, Ruby, Java, C, C++, C#, Objective-C, CoffeeScript, TypeScript, Coldfusion, and SQL in Atom。详细请戳https://atom.io/packages/atom-beautify

5：autoprefixer：写css时由于某些属性要加浏览器内核前缀，自己一个个敲非常麻烦有没有。这个插件就能格尼自动补全。Prefix CSS and SCSS。。详细请戳https://atom.io/packages/autoprefixer

6：docblockr：写代码的一个好习惯就是写注释。不写注释一段时间再去看自己的代码。那就是一个悲剧好吗？你会挖槽，挖槽。这是什么，这个又是什么鬼。。哈哈哈！A helper package for writing documentation。。详细请戳https://atom.io/packages/docblockr

7：lint:代码校验工具。规范我们的编码。我用它和linter-eslint配合使用。加上.eslintrc的配置文件

8： color-picker。颜色取色器。详细请戳https://atom.io/packages/color-picker

9:minimap :就是Sublime右边那一竖块，显示缩小版的代码

10:git-plus。。好像挺不错的，不过我还没用。我要去试一试。

11：react和react-snippets:在atom中写es6和react非常爽。。这我也我大爱它的主要原因。详细请戳https://atom.io/packages/react和最重要的https://orktes.github.io/atom-react/#snippets-list 心情简直不能再愉快。

装逼必备

1：这个首选当然是activate-power-mode：简直炫酷。。详细请戳activate-power-mode 亮瞎双眼。哈哈哈

2:file-icons：有了它每一个文件前面都有个漂亮的图标。。心情愉快。所有安装的插件度可以在navbar的packages中找到

总结

目前就写这么多。有新想法了在补充。。以上都是在表扬atom，那它有缺点没。。当然有。哎哎哎。打开速度和sublime简直不能比好吗？？有时候还卡住。。cpu蹭蹭往上涨。当然也可能是我电脑配置不行。难以言说的痛。不过这些缺点根本不妨碍我爱它。而且我相信它会越开越好。越来越好。。来来来。哈哈哈
