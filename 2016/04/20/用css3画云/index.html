用css3画云

发表于 2016-04-20   |   分类于 css/css3

前言

今天用css3的属性来画一朵云

html结构

首先肯定是html结构啦！这就像建房子一样。这就是毛坯房。

<div id="cloud"></div>

结构非常简单

css

#cloud {
    width: 350px;
    height: 120px;
    background: #f2f9fe;
    background: linear-gradient(top, #f2f9fe 5%, #d6f0fd 100%);
    border-radius: 100px;
    -webkit-border-radius: 100px;
    -moz-border-radius: 100px;
    position: relative;
    margin: 120px auto 20px;
}

当然如果要在主流浏览器中都有效果。肯定少不了兼容啦！你可以自己全部写。也可以利用一些兼容的js文件

#cloud:after,
#cloud:before {
    content: '';
    position: absolute;
    background: #f2f9fe;
    z-index: -1;
}
#cloud:after {
    width: 100px;
    height: 100px;
    top: -50px;
    left: 50px;
    border-radius: 100px;
    -webkit-border-radius: 100px;
    -moz-border-radius: 100px;
}
#cloud:before {
    width: 180px;
    height: 180px;
    top: -90px;
    right: 50px;
    border-radius: 200px;
    -webkit-border-radius: 200px;
    -moz-border-radius: 200px;
}

用css3的伪元素给div添加东西。设置div为position:relative。给这2个伪元素设置position:absolute。然后让它们相对于div定位。即可、、这里的伪元素没有设置display。是因为设置position。进而产生BFC
